# Sistema de Análise de Dados de Pedidos

- Este projeto é um sistema em Python que conecta a um banco de dados PostgreSQL, realiza análises automáticas e gera gráficos com insights sobre vendas e produtos.

## Funcionalidades
- Conexão com banco de dados PostgreSQL
- Leitura de dados das tabelas orders e orderItems
- Junção (JOIN) entre pedidos e produtos
- Análise automática de dados
- Geração de gráficos
- Detecção de outliers (valores fora do padrão)
- Identificação dos produtos mais vendidos
- Análises realizadas
- Vendas ao longo do tempo
- Total de vendas por dia
- Total de faturamento
- Soma total dos pedidos (corrigido para evitar duplicação)
- Produtos mais vendidos
- Ranking dos produtos com maior quantidade vendida
- Outliers
- Identificação de pedidos com valores fora do padrão
## Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
- SQLAlchemy
- psycopg2
- python-dotenv
